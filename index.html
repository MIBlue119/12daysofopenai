Today's Major Updates

🔄 Reinforcement Fine-tuning (RFT)

  • • Advanced from "high school to PhD level" reasoning
  • • Achieves significant results with just 12 examples
  • • Same technique used in GPT-4 and O1 series training
  • • Public launch scheduled for early 2025
  • • Implemented in Thompson Reuters' Co-counsel AI

⚙️ Technical Implementation

  • • JSONL format training data structure
  • • Separate validation dataset prevents memorization
  • • Advanced grading system (0-1 score range)
  • • Distributed training infrastructure support
  • • Automated hyperparameter optimization

🧬 Berkeley Lab Case Study

  • • 300 million people affected by rare diseases globally
  • • 1,100 training examples from medical literature
  • • 31% Top-1 accuracy (Fine-tuned O1 Mini)
  • • Surpassed base O1 performance (25%)
  • • Novel open-ended symptom querying capability

Today's Major Updates

🎥 Sora Release Highlights

  • • Now available for ChatGPT Plus/Pro users
  • • Supports multiple generation modes including text-to-video and image-to-video
  • • Maximum 20-second videos at 1080p resolution
  • • Built-in community exploration features
  • • Currently not available in EU/UK regions

⚙️ Core Features

  • • Text-to-Video generation with detailed scene descriptions
  • • Image-to-Video animation with automatic scene analysis
  • • Storyboard editor with timeline-based scene arrangement
  • • Blend effects for unique transitions
  • • Smart loop creation for seamless animations

🔑 Access Permissions

  • • ChatGPT Plus: 50 generations per month, standard queue
  • • ChatGPT Pro: Unlimited slow generations, 500 fast generations
  • • Limited high-resolution generations for Pro users
  • • Community gallery access for all users

🧪 Technical Highlights

  • • Powered by Sora Turbo accelerated model
  • • Complete physics world simulation capability
  • • Multiple resolution and length settings
  • • Built-in safety review mechanisms
  • • Ongoing optimization for speed and cost
